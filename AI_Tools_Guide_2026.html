2. Claude Code (Terminal AI)

Anthropic Pro/Max Plan Claude Code

AI assistant that runs directly in your terminal with full access to your file system, git, and development tools.

Installation

# Install via npm (Node.js required)
npm install -g @anthropic-ai/claude-code

# Or via Homebrew (macOS)
brew install claude-code

# Login
claude login

Key Features

  • File Operations: Read, write, edit any file in your project
  • Git Integration: Commits, branches, PRs, all via natural language
  • Code Generation: Write entire features, fix bugs, refactor
  • Multi-file Editing: Changes across your entire codebase
  • Web Search: Research while coding
  • Task Management: Built-in todo tracking

Common Commands

# Start Claude Code in current directory
claude

# Start with a specific task
claude "fix the bug in auth.js"

# Resume previous conversation
claude --resume

# Use specific model
claude --model opus

Slash Commands

CommandFunction
/helpShow available commands
/clearClear conversation
/commitCreate git commit from changes
/reviewCode review current changes
/costShow token usage
Pro Tip: Claude Code can run multiple "agents" in parallel for complex tasks like research across many files simultaneously.

3. Claude Co-Work

Anthropic Max Plan Co-Work Mode

Background AI assistant that works autonomously on tasks while you do other things.

How It Works

┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐ │ You assign │────▶│ Claude works │────▶│ You review │ │ a task │ │ in background │ │ results │ └─────────────────┘ └─────────────────┘ └─────────────────┘ │ ▼ Notifications when complete

Best Use Cases

  • Research: "Research all competitors in the spare parts market"
  • Code Review: "Review all PRs from this week"
  • Documentation: "Document all API endpoints"
  • Data Analysis: "Analyze sales data and create a report"

Example Session

# In Claude.ai with Max plan
You: "I need you to research Partao, SPARETECH, and Partium
     competitors. Find their funding, team, and recent news.
     Work on this in the background."

Claude: "I'll work on this research task. You'll be notified
        when I've completed the analysis."

[Claude researches autonomously for 10-15 minutes]

Claude: "Research complete. Here's what I found..."

5. Model Context Protocol (MCP)

Anthropic MCP - Universal AI Connector

Open protocol that allows AI assistants to connect to external tools, databases, and services.

What is MCP?

Think of MCP as a universal translator between AI and external tools. Instead of building custom integrations for every service, MCP provides a standard way for AI to:

  • Read from databases
  • Call APIs
  • Execute workflows
  • Access files and services

Architecture

┌─────────────────┐ │ Claude AI │ │ (or any LLM) │ └────────┬────────┘ │ MCP Protocol ▼ ┌─────────────────┐ │ MCP Server │ ◄── Node.js/Python server └────────┬────────┘ │ ┌────┴────┬────────┬────────┐ ▼ ▼ ▼ ▼ ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐ │ N8N │ │Notion │ │Google │ │ Any │ │ │ │ │ │ Drive │ │ API │ └───────┘ └───────┘ └───────┘ └───────┘

Configuration (Claude Desktop)

MCP servers are configured in: ~/Library/Application Support/Claude/claude_desktop_config.json

{
  "mcpServers": {
    "n8n-mcp": {
      "command": "node",
      "args": ["/path/to/n8n-mcp/dist/mcp/index.js"],
      "env": {
        "N8N_API_URL": "https://your-n8n.app.n8n.cloud",
        "N8N_API_KEY": "your-api-key"
      }
    },
    "google-drive": {
      "command": "node",
      "args": ["/path/to/google-drive-mcp/build/index.js"]
    }
  }
}

6. N8N Automation

Automation N8N Workflow Automation

Open-source workflow automation tool. Like Zapier but self-hostable and more powerful.

URL: n8n.io

Key Concepts

ConceptDescription
WorkflowA sequence of connected nodes that process data
NodeA single action (HTTP request, send email, etc.)
TriggerWhat starts the workflow (schedule, webhook, etc.)
CredentialsSaved authentication for services (API keys, OAuth)

Useful Nodes

NodePurpose
Schedule TriggerRun on schedule (cron)
HTTP RequestCall any API
RSS Feed ReadFetch news feeds
CodeCustom JavaScript
OpenAI / PerplexityAI integrations
Gmail / EmailSend emails
MergeCombine data from multiple sources
